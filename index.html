Raynor College of Massage & Natural Therapies

Design - Wordpress Development

Brandon Raynor the founder of Raynor College of Massage & Natural Therapies is a long term client of 5+ years who I manage multiple websites for including a WordPress multisite.

This particular project required a design refresh, content update, and migration from a multisite installation.

A new design was done to update the school's online brand and better present the course information as well as allow for online course enrolments. This included working with a copywriter to update the website content and improve readability and presentation.

A key component of this job was adding the ability to enrol and pay for upcoming courses with the unique requirement that each course would be sold in one particular currency and not be converted at checkout.

Additionally we wanted to be able to list upcoming courses chronologically. In order to achieve this I used WooCommerce to be able to create and sell the courses, and combined with ACF I was able to display those upcoming courses in relevant ares of the site, and create a filterable upcoming course schedule.

Additional features include: products being auto disabled after the course date passes, date programmable special prices such as Early Bird specials etc., setting course details such as start and end date, course teacher, and more.

As a later addon, a school graduate public directory was added which lists graduates of Raynor College of Massage & Natural Therapies. Using FacetWP and WordPress' custom post type meant I could create a browsable map which filtered graduates based on location, in a very short amount of time.

Project Snapshot

Full Website redesign and content update.

Migrated from multisite installation to clean WordPress build using Bedrock.

Theme built from the groupd up with Sage 8, ACF.

Course sales via WooCommerce. Required custom integration to allow selling different courses in different currencies without currency conversion.

Added the ability to display WooCommerce products as upcoming courses and filter by date, course type, etc.

Browsable Graduates directory/finder using Google Maps and location filtering with the help of FacetWP.

Managed copywriter to update website content.
